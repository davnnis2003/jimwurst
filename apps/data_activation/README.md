# Data Activation

Data Activation is the layer where data is transformed into value through visualization, analysis, and automated action. This component provides the tools and interfaces for business stakeholders and systems to consume processed data.

## Scope

- **Reporting**: Production-grade dashboards and automated reports for tracking key business metrics.
- **Ad-hoc Analysis**: Flexible environments for exploratory data analysis and rapid hypothesis testing.
- **Machine Learning (ML)**: Deploying models for predictive analytics, classification, and advanced insights.
- **Artificial Intelligence (AI)**: Leveraging Generative AI (e.g., local execution via **Ollama**) to enable natural language interaction with data and intelligent automation.

## Structure

- **metabase/**: Configuration for Metabase dashboards and connections.
